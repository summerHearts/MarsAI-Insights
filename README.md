# Mars模型评测平台

Mars模型评测平台是一个用于大型语言模型评测、比较和分析的综合工具，旨在帮助用户轻松评估不同大语言模型的性能、质量和适用性。该平台支持多种模型API的调用、结果比较、批量数据处理和可视化分析等功能。

![Mars模型评测平台](./public/logo192.png)

## 功能特性

### 模型比较
- 支持同时调用多个AI模型进行内容生成比较
- 可配置提示词模板，确保评测一致性
- 支持文本预处理，包括时间戳移除、HTML标签清理、空白规范化等
- 提供角色增强功能，优化会话型数据的处理
- 支持保存比较结果，方便后续分析和对比

### 批量数据评测
- Excel文件批量处理功能，支持大规模数据集评测
- 配置灵活的并发控制和批处理参数
- 处理过程可视化，支持暂停和恢复
- 结果自动保存和导出
- 支持处理完成后的飞书通知

### 高级分析工具
- 结果评测：专门用于评估大模型判责结果的合理性
- 数据分析与图表：生成可视化图表，直观展示模型性能差异
- Excel处理工具：便捷处理表格数据

### 系统配置
- 模型配置管理：灵活配置多种模型参数和API接口信息
- 提示词模板管理：创建、编辑和管理常用提示词
- 系统通知配置：支持配置飞书通知，实时获取处理结果
- 历史记录管理：查看、导出和导入历史评测结果

## 模块详细介绍

### 1. 模型比较
![模型比较功能](./public/assets/01.png)

模型比较功能允许用户同时调用多个AI模型，对同一个提示词或输入进行处理，直观对比不同模型的回答质量、速度和准确性。
- **多模型选择**：支持同时选择多个已配置的模型进行对比
- **提示词模板**：可以使用预设的提示词模板，确保对比的一致性
- **实时响应对比**：直观展示各个模型的响应时间和内容差异
- **保存比较结果**：可将比较结果保存到历史记录中，便于后续查看和分析

### 2. 批量数据评测
![批量数据评测](./public/assets/02.png)

批量数据评测模块支持用户上传Excel文件，对大量数据进行自动化处理，是进行大规模模型评测的理想工具。
- **Excel文件导入**：支持上传.xlsx和.xls格式的Excel文件
- **灵活列选择**：可自定义输入列和输出列，灵活配置处理逻辑
- **进度可视化**：实时显示处理进度，支持暂停和恢复功能
- **结果导出**：处理完成后可一键导出为Excel文件
- **批处理参数调优**：可调整并发数、重试次数等参数，优化处理效率

### 3. 提示词模板
![提示词模板](./public/assets/03.png)

提示词模板管理模块允许用户创建、编辑和管理常用的提示词模板，提高评测效率和一致性。
- **模板分类管理**：按用途和功能对模板进行分类管理
- **变量支持**：模板支持变量插入，增强灵活性
- **一键应用**：可在模型比较和批量处理中一键应用模板
- **导入导出**：支持模板的批量导入和导出，便于团队共享

### 4. Excel处理
![Excel处理](./public/assets/04.png)

Excel处理模块提供了便捷的表格数据处理功能，支持数据清洗、转换和预处理。
- **数据清洗**：自动识别并处理缺失值、异常值
- **列操作**：支持增加、删除、重命名列等操作
- **数据转换**：支持文本清理、格式转换等功能
- **预览功能**：实时预览处理结果，确保数据质量

### 5. 模型处理结果评测
![结果评测](./public/assets/05.png)

模型处理结果评测模块专门用于评估大模型判责结果的合理性和准确性，支持专家规则验证。
- **规则设置**：支持设置评测规则和标准
- **准确率分析**：自动计算判断准确率和一致性
- **结果可视化**：直观展示评测结果和统计数据
- **人工审核**：支持对模型判断结果进行人工审核和标注

### 6. 数据分析与图表
![数据分析与图表](./public/assets/06.png)

数据分析与图表模块提供强大的可视化能力，将评测数据转化为直观图表，帮助用户理解模型性能差异。
- **多维数据分析**：支持从多个维度分析模型性能
- **自定义图表**：可选择不同类型的图表呈现数据
- **交互式图表**：支持缩放、筛选等交互操作
- **图表导出**：支持导出高质量图表用于报告和演示

### 7. 模型配置
![模型配置](./public/assets/07.png)

模型配置模块允许用户灵活配置各种模型参数和API接口信息，支持多种模型的接入和管理。
- **多模型管理**：支持添加、编辑和删除多个模型配置
- **参数调优**：可调整温度、top_p等生成参数
- **API配置**：支持配置端点URL、密钥和请求格式
- **高级设置**：提供自定义请求头、超时等高级配置选项

### 8. 系统通知配置
![系统通知配置](./public/assets/08.png)

系统通知配置模块支持对接飞书等通知系统，当批处理任务完成或出现异常时自动发送通知。
- **飞书机器人对接**：支持配置飞书机器人Webhook
- **通知模板**：可自定义通知内容和格式
- **事件触发**：灵活设置通知触发条件
- **安全设置**：支持签名密钥等安全配置

## 技术栈

- **前端框架**：React 19 + TypeScript
- **UI组件库**：Ant Design 5
- **图表可视化**：ECharts
- **HTTP请求**：Axios
- **Excel处理**：xlsx
- **语法高亮**：react-syntax-highlighter
- **文件处理**：file-saver

## 安装和使用

### 安装依赖

```bash
# 克隆项目
git clone https://github.com/yourusername/mars-model-evaluation.git
cd mars-model-evaluation

# 安装依赖
npm install
```

### 配置环境变量

创建`.env`文件，或复制`.env.example`并根据需要进行修改：

```
REACT_APP_DEFAULT_API_KEY=your_default_api_key
```

### 启动开发服务器

```bash
# 默认端口启动
npm start

# 指定端口启动
npm run start:port3001  # 在端口3001启动
npm run start:port8080  # 在端口8080启动

# 交互式启动（选择端口）
npm run start:interactive
```

### 构建生产版本

```bash
npm run build
```

生成的文件将位于`build/`目录下，可以部署到任何静态文件服务器。

## 使用指南

### 模型配置

1. 点击导航栏中的"模型配置"
2. 添加新模型或编辑现有模型
3. 配置模型名称、API端点、请求模板等参数
4. 支持高级设置如请求头、强制模型名称等

### 模型比较

1. 选择要比较的模型
2. 输入提示词和待处理文本
3. 可选择启用文本预处理
4. 点击"开始比较"按钮获取结果
5. 比较结果可保存至历史记录

### 批量数据评测

1. 上传Excel文件（支持.xlsx和.xls格式）
2. 选择模型、输入列和输出列
3. 配置并发数和处理参数
4. 启动处理并等待结果
5. 可导出结果为Excel文件

### 系统通知配置

1. 配置飞书机器人Webhook和密钥
2. 启用通知功能
3. 批处理完成后自动发送通知到飞书

## 本地存储说明

应用使用浏览器的localStorage存储以下数据：
- 模型配置
- 提示词模板
- 保存的比较结果
- 保存的输入内容
- 批处理任务
- 飞书通知配置

对于大量数据处理，系统会自动优化存储以避免超出浏览器限制。

## 注意事项

- 本应用依赖浏览器localStorage存储数据，清除浏览器缓存可能导致数据丢失
- 处理大规模数据时，建议使用较低的并发数，避免API限流
- 对于重要数据，请使用导出功能定期备份
